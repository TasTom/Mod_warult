# Expedition 33 Faction

Une faction mystérieuse pour RimWorld inspirée de Clair Obscur: Expedition 33.

## Installation
1. Abonnez-vous sur Steam Workshop
2. Activez le mod dans RimWorld
3. Redémarrez le jeu

## Fonctionnalités
- Faction Expédition 33 avec équipements uniques
- Événement "Le Gommage" 
- Bâtiments spécialisés
- Système de missions

## Compatibilité
- RimWorld 1.4+
- Compatible avec tous les DLC
- Aucune dépendance requise
